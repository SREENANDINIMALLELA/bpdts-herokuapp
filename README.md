
### Description
A maven java project with API to return users with in the given miles of a city



## Dependencies
 - Maven
 - Spring Boot web , test
 - Logback
 - Junit

## Running Locally

Setup
```
mvn clean install
```
Run
```
mvn spring-boot:run
```

### Calling API
Postman (GET)
```
localhost:8080/city/London
```

## Testing
Run
```
mvn clean test
```
